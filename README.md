## Installation and Setup

### Backend Installation and Setup

Navigate to the Backend Directory:

```bash
  cd backend
  npm install
  npm run dev
```

### Frontend Installation and Setup

Navigate to the Frontend Directory:

```bash
  cd frontend
  npm install
  npm run dev
```
